## ğŸ“± Features
This app helps Mobile Legends players dominate the draft phase with a highly optimized, overlay-ready interface.

* **Real-time Counter Picks:** Algorithm to suggest the best heroes against the enemy team.
* **Smart Drafting:** Suggestions based on team composition.
* **Hard Responsive UI:** Specifically designed to work as a **Floating Window/Overlay** while playing, without breaking the layout or dropping frames.
* **Serverless Backend:** High availability and speed without the need for traditional server maintenance.
